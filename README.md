## 🇮🇹 Italiano

### Descrizione
**Torre di Ingegneria** è un vertical climber arcade basato sull'iconica torre della facoltà di Ingegneria dell'Università Politecnica delle Marche (Ancona). Il giocatore deve scalare la torre saltando su piattaforme procedurali, sfidando la gravità e gli agenti atmosferici per raggiungere l'altezza massima.

### Caratteristiche
- **Salita Infinita:** Piattaforme generate proceduralmente con diverse proprietà (normali, fragili, elastiche, bonus Gulliver).
- **Sfondo Dinamico:** Ciclo giorno/notte dinamico che passa dal cielo azzurro allo spazio profondo man mano che si sale.
- **Sistema Meteo:** Effetti procedurali di pioggia e neve alle basse quote.
- **Design Brutalista:** Logo e grafica ispirati all'architettura reale del polo di Montedago.
- **Ottimizzato per Studenti:** UI moderna in stile glassmorphism con richiami al brand UNIVPM.

### Tecnologie
- [Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API) - Rendering grafico 2D ad alte prestazioni.
- [Vite](https://vitejs.dev/) - Build tool e ambiente di sviluppo moderno.
- JavaScript ES6 (Modulare e procedurale).
- CSS Moderno (Glassmorphism e animazioni fluide).

### Come giocare (Locale)
1. Installa le dipendenze:
   ```bash
   npm install
   ```
2. Avvia il server di sviluppo:
   ```bash
   npm run dev
   ```

## 🇬🇧 English

### Description
**Engineering Tower** is an arcade vertical climber based on the iconic engineering faculty tower at the Università Politecnica delle Marche (Ancona). The player must climb the tower by jumping on procedural platforms, defying gravity and weather conditions to reach the highest altitude possible.

### Features
- **Endless Climb:** Procedurally generated platforms with different properties (standard, crumbling, bouncy, Gulliver bonus).
- **Dynamic Background:** Day/night cycle that transitions from blue sky to deep space as you climb.
- **Weather System:** Procedural rain and snow effects at lower altitudes.
- **Brutalist Design:** Logo and graphics inspired by the real architecture of the Montedago campus.
- **Student Optimized:** Modern glassmorphism UI with UNIVPM branding elements.

### Tech Stack
- [Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API) - High-performance 2D rendering.
- [Vite](https://vitejs.dev/) - Modern build tool and dev environment.
- Modular JavaScript ES6.
- Modern CSS (Glassmorphism and fluid animations).

### How to Play (Locally)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
